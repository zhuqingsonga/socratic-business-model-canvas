# Socratic Business Model Canvas - 苏格拉底式商业模式画布

Guide you to systematically explore and build a complete Business Model Canvas through Socratic deep questioning.

用苏格拉底式深度追问的方式，引导你系统地探索和构建完整的商业模式画布。

## Core Features - 核心特点

### 🧠 Socratic Questioning - 苏格拉底式追问
- Instead of directly filling out 9 blocks, help you think deeply about the logic behind each block through progressive questions
- Gentle guided exploration style, not challenging assumptions
- Start shallow, go deep, let you discover answers yourself

- 不是直接填写9个模块，而是通过层层递进的问题，帮助你深度思考每个模块背后的逻辑
- 温和的引导探索风格，而非挑战假设
- 从浅入深，让你自己发现答案

### 🎯 Target Users - 目标用户
- **Product Managers - 产品经理**：梳理产品商业模式
- **Founders - 创业者**：设计和验证创业项目
- **Executives - 企业高管**：战略规划和业务梳理

### 📊 Complete Coverage - 完整覆盖
- Customer Segments - 客户细分
- Value Propositions - 价值主张
- Channels - 渠道通路
- Customer Relationships - 客户关系
- Revenue Streams - 收入来源
- Key Resources - 核心资源
- Key Activities - 关键业务
- Key Partnerships - 重要合作
- Cost Structure - 成本结构

## How to Use - 使用方式

Trigger this skill in OpenClaw, then follow the questions step by step to explore your business model.

在 OpenClaw 中触发这个 skill，然后跟着问题一步步探索你的商业模式即可。

## Design Philosophy - 设计理念

> "I cannot teach anybody anything, I can only make them think." —— Socrates
> 
> "我不能教给任何人任何东西，我只能让他们思考。" —— 苏格拉底

This skill is designed to help you discover the answers about your business model through deep thinking, rather than giving you a template directly.

这个 skill 的设计思路是：帮助你通过深度思考，自己发现商业模式的答案，而不是直接给你一个模板。
